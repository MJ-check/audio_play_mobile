# Audio Play Mobile

## 开放音乐平台移动端项目

&emsp;&emsp;开放音乐平台移动端项目。项目使用 nodejs + React 搭建，目前版本 V1.1 ，后期根据需求更新。

## 项目部署

1. 克隆项目到本地

   ```powershell
   git clone https://github.com/MJ-check/audio_play_mobile.git
   ```

2. 安装依赖

   ```powershell
   npm install
   // 或者使用 cnpm
   cnpm install
   ```

3. 运行项目

   ```powershell
   npm run start
   ```

   - 运行项目之前需要先启动后台服务。后台服务项目链接地址：

     <https://github.com/MJ-check/audio_play_server>

4. 打包项目

   ```powershell
   npm run build
   ```
